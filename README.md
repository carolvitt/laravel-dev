## Sobre o projeto

Aplicação com sistema de registro e login, com o objetivo de mostrar algumas Universidades dos Estados Unidos e possibilitar a inscrição do usuário nestas.
Para tanto, as tecnologias utilizadas foram:

- PHP.
- Bootstrap.
- JQuery.
- Requisições ajax.
- Linguagem MySQL.
